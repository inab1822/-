## 예제 1. 상하좌우
- 여행가 A는 N X N 크기의 정사각형에 있다.
- 이 공간은 1 X 1 크기 정사각형으로 나누어져 있다.
- 가장 위 좌표 (1,1), 가장 오른쪽 아래좌표 (N,N)

- L R U D
- left, rigth, up, down
- 각각 방향대로 한칸씩 이동
- 정사각형 공간을 벗어나는 움직임은 무시
- 명령(L R U D)이 주어집니다.
- 이때 도착하는 공간의 좌표를 구하세요.

[입력조건]
- 첫째 줄 : 공간 크기를 나타내는 N
- 둘째 줄 : 이동 명령이 주어집니다.

[출력조건]
- 좌표 X(행), Y(열)

[예시]
```
5
R R R U D D
```
```
x x x x x
x x x x x
x x x x x
x x x x x
x x x x x
```
